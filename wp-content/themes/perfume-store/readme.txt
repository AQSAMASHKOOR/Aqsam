== Description ==

The Perfume Store WordPress Theme is a sleek, elegant, and feature-rich solution designed for perfume shops, fragrance stores, and luxury perfume boutiques looking to establish a strong online presence. Whether you run a niche perfume store, a designer perfume store, or an online perfume store specializing in exclusive scents, this theme provides a perfect balance of style and functionality. It is crafted with responsive design, ensuring a seamless browsing experience across all devices. With SEO-friendly coding and optimized performance, your store can rank higher in search results, attracting fragrance lovers worldwide. The theme is perfect for showcasing best-selling perfumes, new arrivals, rare perfume collections, and limited edition perfumes, making it ideal for both luxury fragrance stores and discount perfume stores alike. Integrated e-commerce features, such as secure payment gateways, cart management, and wish list options, make shopping convenient and enjoyable for customers. Visually, the theme exudes sophistication with a stunning homepage banner, elegant typography, and strategically placed Call to Action (CTA) buttons to drive conversions. Dedicated sections for perfume gift sets, seasonal perfume sales, and fragrance recommendations enhance user engagement. The theme also supports perfume subscription stores, allowing customers to receive curated scents regularly. Social media integration lets customers share their favorite fragrances, increasing brand visibility.

== Homepage Installation ==

Step-1. Go to Dashboard -- Pages -- Add new page -- Add templates (Home Page) -- Publish it.
Step-2. Go to Dashboard -- Settings -- Reading -- Click on static page -- Select home page -- Save changes.

Header Section
Step-1. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Header Sidebar Section -- Add Text -- Add Mail Title -- Add Email Address -- Add Phone Title -- Add Phone Number -- Add links -- Publish it.

Banner Section
Step-1. Go to Dashboard -- Pages -- Add New Page -- Add title, content, and featured image -- Publish it.
Step-2. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Banner Section -- Check To Enable This Section -- Select Page to display Banner -- Add Banner Small Title -- Add Banner Button Text -- Publish it.

Trending Products Section
Step-1. Go to Dashboard -- Products -- Categories -- Add Name -- Add New Category.
Step-2. Go to Dashboard -- Products -- Add New Products -- Add title, content, price and featured image -- Select Category -- Publish it.
Step-3. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Trending Products Section -- Check To Enable Section -- Add Trending Products Title -- Select category to display products -- Publish it.

Footer Section
Step-1. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Footer Section -- Add copyright text -- Publish it.
